Problem - C - Codeforces

题目大意:给定数p和q,请你找出最大的数x,使得x是p的因数,且q不是x的因数.

思路:数论题.如果p不能被q整除,那么q就不是p的因数,那么最大的数x=p.如果可以被整除,那我们只需要求出一个最小值,当p除以这个最小值时,q不再是p的因数即可.例如4是p的因数,也是q的因数,那么如果将p除以2以后,q就比p多了一个因数2,q就不再是p的因数了,所以我们需要做的就是找出p和q的所有因数,然后对每个因数进行枚举就可以了.

Problem - D - Codeforces

Problem - A - Codeforces

题目大意:给定数a和b,请你找出一个数x,使得a^x + b^x的值最小,输出最小值.

思路:为了使两个数异或之后的和尽可能地小,那么只有当两个数同一位上都是1的时候才会变小,此时x该位上可以为1,如果a,b该位上不相等,那么x为1或者为0都不影响结果.那么可以发现,a^x+b^x的最小值就是a^b的值.

Problem - C - Codeforces

题目大意:给定你一个字符串,你每次可以进行两种操作,从选取子串a2, a3, …ai(i<n)然后将该子序列翻转加到原字符串右边,或者选取子串ai, ai+1,…an-1将其逆置后加到原字符串的右边,你最多可以进行30次这样的操作,请输出一种处理方案使得变换后得到的字符串为回文串.

思路:仔细观察可以得到,每次进行操作一就相当于将区间[2,i]的部分以1为对称轴对称到左边,操作二就相当于以n为对称轴对称到右边.为了得到回文串,那么我们只需要将原串[2,n-1]的部分对称到右边,然后再补齐左端点的值就可以了.但是因为我们每次操作不能将两边的字符添加到字符串中,那么我们想要进行这样的构造,则需要满足一个条件,当前字符串的第一个字符等于第三个字符,这样进行对称操作后,倒数第二个字符就等于左端点字符,就可以将其加入字符串得到回文串,为了满足第一种情况只需要先进行一次操作一将第二个字符加到左边即可.
