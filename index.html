Problem - C - Codeforces

题目大意:给定数p和q,请你找出最大的数x,使得x是p的因数,且q不是x的因数.

思路:数论题.如果p不能被q整除,那么q就不是p的因数,那么最大的数x=p.如果可以被整除,那我们只需要求出一个最小值,当p除以这个最小值时,q不再是p的因数即可.例如4是p的因数,也是q的因数,那么如果将p除以2以后,q就比p多了一个因数2,q就不再是p的因数了,所以我们需要做的就是找出p和q的所有因数,然后对每个因数进行枚举就可以了.

Problem - C - Codeforces

题目大意:给定数n,k,你可以选择n个小于2^k-1的数使得a1&a2&a3&…&an >= a1^ a2^a3^…^an,请问有多少种选择方式,相同的数在不同的位置上也算不同的种类.

思路:组合数学+位运算.首先我们考虑一位的情况.

如果两个数相等,如果想要与操作的结果为1,那么n个数都为1,此时如果是奇数,那么异或操作的结果也为1,符合.如果不是n个1进行与操作,那么与操作得到的数一定是0,那么为了使得相等,我们必须让异或操作也为0,那么对于这种情况下只能选择偶数个1.

在考虑与操作大于异或操作的情况,这种情况下,只能是与操作为1,异或操作为0,也就是n为偶数,且有n个1的情况,这种情况下如果当前位大于了,那么剩下的低位可以随意的组合,对于这种情况我们只需要从高位开始枚举相等的那一位,该位之前的种类就是相等的组合数^当前相等的位数,该位之后的种类就是任意组合也就是(2^n)^剩余低位的个数.

思路:比较简单的思维题,因为大于2的质数一定是奇数,那么它对于2的余数一定是1,然后对于n,它和n-1互质,与n-1的余数也是1,所以只需要输出2和n-1即可.

Problem - B - Codeforces

题目大意:给定一张n*n的棋盘,Alice的棋子都排放在最下面一列,敌人的棋子都摆放在最上面一列,现在敌人不移动棋子,Alice可以按照以下规则来移动棋子:

  1. 如果该棋子上方无其他棋子,可以直接向上移动一格.
  2. 如果该棋子斜上方有一个敌人的棋子,可以直接移动到敌人的位置.

请问Alice最多有多少颗棋子可以移动到最上方一行.

思路:模拟,其实无论棋盘有多长都无关,实际上Alice所所有的棋子和敌人的棋子可以抽象成两行,一行最上方敌人的棋子,一行是Alice的棋子,在第二行,为了使棋子尽可能地上移,那么对于第一行为空的位置,只能由第二行补上,而有敌人的位置,需要从左下,右下选择一个到达,只需要简单模拟一下即可.

Problem - C - Codeforces

题目大意:给定n个人,每个人权力等于他的编号,然后若干个人之间会有友谊关系.你可以进行一下三种操作:

  1. 让某两个人之间建立友谊关系.
  2. 清除某两个人之间的友谊关系.
  3. 计算当前的存活人数,计算规则如下,如果一个人的权力低于他的每个朋友,那么该人被清除,该过程会一直进行下去,直到停下为止,输出停下后还存活的人数.(清除会从权力最小的人开始清除,不是同时清除)

现在进行若干次操作,请对于每个操作进行处理.

思路:思维题,抽象成图就是,如果一个节点连接了比他权力大的节点,那么这个节点一定会被清除掉,那只需要线性维护一下每个人所连接的权利大于他的节点数量即可.

Problem - D - Codeforces

题目大意:如果一个序列满足,对于序列中的每一个数,存在一个数m,满足ai%m都相等,那么我们称这样一个序列是一个good序列,对于一个长度为n的序列,请你求出它的子序列中good序列的最大长度.

思路:因为要求ai%m相等,ai = m*j + i, i是余数那么我们只需要考虑m*j即可,实际上就是求区间上每个m*j的gcd,那么我们需要做的就是用差分数组维护一个gcd,然后再枚举左端点,因为gcd满足单调性,对于区间长度固定后,如果增加区间长度,gcd只可能变小,而减小区间长度,gcd可能变大,所以对于右端点可以采用二分来选取.这样复杂度就是nlogn+nlogn.
