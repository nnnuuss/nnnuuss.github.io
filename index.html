Problem - C - Codeforces

Problem - A - Codeforces

题目大意：给你a个物品x，b个物品y，你每次可以消耗2个x，一个y造一个物品或者一个x，2个y来造一个物品，请你对于给定的a，b求出最多可以构造的物品数量。

思路：思维题，对于a+b/3构造得到的是理想状态下能够构造得到的物品个数，对于min（a，b）是每次只消耗最少的物品来合成物品的构造方式，对两种方式取最小的解才能得到合法解。

Problem - B - Codeforces

题目大意：给你长度为n的序列，序列中只有一个数为1其余都为0，给定你值为1的数的位置，然后给你若干组数据l**i,ri,你每次可以随意交换区间[li,ri**]内的元素，请你判断出最后有多少个位置的值可能为1.

思路：思维题，每次维护一个x可以到达的区间，如果给定的区间与x的区间有重合部分，那么就进行区间扩展，最后扩展得到的区间就是答案。

Problem - C - Codeforces

题目大意：给定一张大小为n*m仅由0，1组成的地图，你需要从左上角走到右下角，每次你只能向下或者向左移动一位，你需要保证从（1，1）到达（n， m）的路径为回文串。

思路：模拟题，直接通过dfs或者bfs，把第几步能到的地方找出来，从左上角到达右下角所需的步数为step=n+m-1，那么每次距离为i和step-i+1的格子必须是相同的，线性扫一次取最小值即可。

Problem - D - Codeforces

题目大意：给定你若干个数，对于每个数能否找到它的两个大于1的因数a1,a2,使得gcd（a1+a2, x）=1。

思路：因为输入规模较大，所以不能直接暴力的拆分，这里需要由数论的一个推论来优化问题，找到一个合适的构造法，首先有一个很重要的结论,如果ab互质，那么ab的乘积和ab之和互质，即gcd(a,b)=1，gcd(a+b,ab)=1，那么对于每个数x如果它有两个及以上的质因子，那么一定可以拆分成a1,a2，对应上面的式子，a和b对应的就是a1,x/a1,为了使这两个数互质，那么x/a1一定不含a1。对于分解质因数的过程还需要用素数筛优化一下。

Problem - E - Codeforces

题目大意：给定长度为n的序列a，长度为m的序列b，你可以将序列a划分成m个子区间使得每个区间内的最小值等于序列b中的每个值，b严格升序排列。

思路：构造题，能够划分的区间内的所有数一定都是大于等于其区间对应的b**i，每次处理需要从大到小来进行处理，因为从大到小能保证每次区间划分时，更大的数会被优化处理，如果从小到大处理，对于大于对应的bi**处理时很难进行区间划分，其余情况就是对开头一段的讨论。

Problem - B - Codeforces

题目大意:给定你l,r,m,现在请你构造出a,b,c满足l<=a,b,c<=r,且n*a+b-c=m.

思路:分两类讨论,一类情况是m%r<l,这类情况下n=m/r,剩余的余数让b-c来补,另一种情况是m%r>l,这种情况下就需要是使得n=(m/r)+1,然后补上负余数.因为l-r<=(b-c)<=r-l.
