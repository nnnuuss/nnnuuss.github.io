Problem - A - Codeforces

题目大意:给定一个数n,你可以选择两个不相等的数a,b,满足(1$\leq$a<b$\leq$n),请你求出最大的GCD(a,b).

思路:不需要想太麻烦,最小的素数是2,那么a和b一个是n可以选择的最大的偶数,一个是偶数/2.只需要特判一下2就可以,因为2和1的gcd为1.

Problem - B - Codeforces

题目大意:给定2n个正数,你每次可以选择两个数进行合并,然后将所有合并的元素作为一个序列,请给出一种合并的方式,使得序列种n-1个元素的gcd不为1.

思路:其实思路很简单,还是最小素数为2的考点,我们只需要将奇数放在一起合并,偶数放在一起合并,那么一定满足这些元素的gcd>=2.

Problem - C - Codeforces

题目大意:给定一个数n,Alice和Bob每次可以对这个数进行两种操作

  1. 将当前数除以该数的一个奇因数.

  2. 将当前数减1(该数为1时不可再进行操作).

Alice先手,如果谁无法进行上述操作了,那么该玩家失败.

思路:找规律,如果当前数是奇数,那么可以直接将该数除以它本身,所以当前数是奇数,先手一定获胜.

如果当前数是偶数,非三类讨论,如果当前数是2,那么一定后手胜,属于一个特判.

如果当前数的因数全为偶数,那么先手只能进行操作2,后手状态转移到奇数状态,后首胜.

如果当前数的因数有奇数,那么先手的赢法就是将该数的变换成非2的偶数,这样后手只能进行操作2,回到奇数状态,先手获胜.

  1. 两个序列都是严格升序的.

  4. 两个序列中只有一个数不相同.

思路:首先一点,只有一个数不相同,且严格升序,当这个不同的数是原序列中最小的值的时候,它就可以在区间[1, a2-1]上除该点的位置任意选取,同理当这个数是最大数的时候,他就可以在区间[an-1+1, k]上除原位置之外的其他位置上选取,当一个点处于两点之间的时候,那么它就只能在两点之间来进行移动了.

因为x / y = k && x % y==k,那么x=(y+1)k

因为k<y, 所以 x > y(y+1), x^1/2 > y > k.

经过这样的一系列证明我们发现k的数据范围降成了1e5左右,那么我们就可以枚举k来求解了.

关于如何枚举k,我们直接计算出a/k的大小,这个计算出来的就是理想下最大的y+1,如果y大于了b,那么y只能为b,对于每个y+1都可以在a的范围内找到对应的x,然后因为k<b,所以有效的b的个数是b-k个,

Problem - C - Codeforces

题目大意:两个人交替参加比赛,共10场,当每场比赛的结果是1的时候就代表参加这场比赛的人获得1分,如果是0则不得分,如果是?则表示待定,现在两人进行比试,请你站在裁判的角度考虑至少比到第几场时,不需要再比了.(至少的意思是:如果当前两人的分差无法通过后续的比赛来进行弥补,那么就不需要再比了,同时你也可以使得?变成0或1来尽快结束)

思路:首先是个对于比赛结果不确定的情况,我们只由两种取法

一种是让奇数场次的不确定场全部获胜,偶数场次的不确定场全部失败

一种是让奇数场次的不确定场全部失败,偶数场次的不确定场全部获胜

只有这样分差才尽可能地大,然后对于结果的判断,只需要统计两个人的分数,然后判断当前的分差是否大于剩余场次能能拉平的分差,以此来判定是否提前结束比赛.

Problem - D - Codeforces

题目大意:你可以操控键盘打出一串预定字符,但在你准备输入一个字符时,你可以选择输入该字符,或者按下删除键(这样你不仅不会输入该字符,还会删掉之前存在的一个字符),现在给你一串预定字符,和一串目标字符,请问你是否可以打出目标字符串.

思路:思维题,首先如果目标串长度大于初始串那么一定不可能.

然后需要讨论两种情况

  1. 目标字符串和源字符串的差值为奇数.
    因为每次删除只由在开头的时候删除,才能删除一个字符,所以如果差值为奇数,那么一定是在开头就进行了一次删除.然后情况就变成了偶数情况

  2. 差值为偶数
    在后续的删除中每次删除就相当于把字符串同时删去了两位,所以对于这种情况只需要线性匹配一次就可以了.

得知这两种情况之后,其实只需要对两个串进行一次匹配,如果失配的话,直接将源字符串左移两位即可,相当于在某个字符失配之后直接进行删除.

思路:模拟题+二分.如果当前情况下我们的分数低于llya,那么在之后添加的每一轮分数都必须等于100,而llya的每一轮分数都必须为0,这样才能使得添加的额外轮的次数最少,然后直接二分枚举出添加的次数,然后对于二分出的答案,用模拟来判断.

Problem - A - Codeforces

题目大意:给定一个数组,每次可选该数组的任意个元素求平均值,对于计算的每个元素,如果其数值大于平均值,那么就将其移出该数组.求出给定数组最多可以删除的元素个数.

思路:对于给定序列,最终能留下的元素,一定是数组中最小的元素,所以只需要统计最小元素的个数,然后拿数组总个数减去最小元素个数即可.

Problem - B - Codeforces

题目大意:给定一个序列,对于序列中任意的i<j,都满足:|ai-aj| >= MAX,(MAX是该序列中最大的元素),那么就乘该序列是一个strange序列,先给定你一个序列,请你求出它最长的strange子序列的长度.

思路:首先对于所有的负数和0,都一定可以添加到strange序列中,因为绝对值一定大于等于0,然后考虑正数的情况,最多只能添加一个正数,因为如果有多个正数,那么正数之差的绝对值一定小于较大的那个正数,对于正数的添加,我们还要考虑正数的大小,如果最小的正数已经大于非正数差值的最小值,那么也无法添加.
