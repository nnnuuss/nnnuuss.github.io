A. Add and Divide

time limit per test

1 second

memory limit per test

256 megabytes

input

standard input

output

standard output

You have two positive integers a and b.

You can perform two kinds of operations:

  • a=⌊a/b⌋ (replace a with the integer part of the division between a and b)
  • b=b+1 (increase b by 11)

Find the minimum number of operations required to make a=0a=0.

Input

The first line contains a single integer t (1≤t≤100) — the number of test cases.

The only line of the description of each test case contains two integers aa, bb (1≤a,b≤1e9).

Output

For each test case, print a single integer: the minimum number of operations required to make a=0.

题目大意:对于a和b有两种操作,一种是对b进行加一,第二种是让a/=b,求最少多少次能使a变为0.

解题思路:起初以为要用数学的思路来做,结果想了半天都给不出一个合理的推论,查了相关题解得知,这道题的输入规模其实完全可以以靠枚举来做,这里给自己的一个启示是,面对大数据数学问题,如果有时候给不出比较好的做法,不如先试试暴力,能跑一部分案例再说.
