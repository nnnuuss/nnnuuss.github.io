A. K-divisible Sum

time limit per test

1 second

memory limit per test

256 megabytes

input

standard input

output

standard output

You are given two integers n and k.

You should create an array of n positive integers a1,a2,…,an such that the sum (a1+a2+⋯+an) is divisible by k and maximum element in a is minimum possible.

What is the minimum possible maximum element in a?

Input

The first line contains a single integer t (1≤t≤1000) — the number of test cases.

The first and only line of each test case contains two integers n and k (1≤n≤109; 1≤k≤109).

Output

For each test case, print one integer — the minimum possible maximum element in array aa such that the sum (a1+⋯+an) is divisible by k.

我的思路是如果k>=n,那么最大值*n就必须大于等于n,顺这个思路就得到了answer = k / n + (k % n != 0)的代码.如果k<n那么就让k等比例的增大.但是超时了,说明这个思路还是太慢了,主要是k<n时的判断重复步骤太多,搜索博客得知,当n>k时如果n能被k整除那么只需让每个a都等于1就可以,如果不行的话,调整若干个a的值,使他们等于2也可以得到答案.
