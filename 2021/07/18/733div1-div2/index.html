Problem - A - Codeforces

题目大意:现在规定一种只由0和1组成的数字,我们称这种数字为二进制数字,例如10,1010111,给定一个数n,求该数字最少由多少个二进制数字组成.

解题思路:每取一个二进制数字,可以使得原数字n上各位都减小1或者0,为了使次数尽可能地小,那么当原数字n上各位不为0的时候都应该-1,那么最小的次数就是各位上最大的数字.

Problem - B - Codeforces

题目大意:给定一个高为h,宽为w的网格,你可以在网格的四个边缘处,放置一个盘子,每个盘子的四周都不能有别的盘子(四周指的是最近的8个格子),请输出一个种安排方式,使得放置的盘子个数尽可能地多.

思路:模拟题,首先为了使个数尽可能多,那么一定是从第一个开始放置,然后检测后面是否合法,如果合法就放下盘子,如果不合法就跳过.

Problem - C - Codeforces

题目大意:你和Ilya在参加一个比赛,比赛分为n轮,每轮取得的分数为0~100之间,现在得知了你和llya完成n轮比赛之后每轮的分数,现在规定你们得分的标准如下:如果你们完成了n轮比赛,那么选取n-[n/4]次所有比赛中最好的成绩相加,得到的和就是你的成绩.([n/4]表示对其向下取整),现在还可以额外添加x轮比赛,求最少添加多少场额外的比赛,(你和llya的分数都不确定)你的分数理论上可以大于等于llya的分数.

思路:模拟题+二分.如果当前情况下我们的分数低于llya,那么在之后添加的每一轮分数都必须等于100,而llya的每一轮分数都必须为0,这样才能使得添加的额外轮的次数最少,然后直接二分枚举出添加的次数,然后对于二分出的答案,用模拟来判断.
