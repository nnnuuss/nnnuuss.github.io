Problem - A - Codeforces

题目大意:判断一个有n条边的正n边形,是否存在两条垂直边.

思路:最小的拥有垂直边的是四边形,在此基础之上每次只能+4条边,才不影响原本存在的垂直边,所以只需要判断n是否被4整除.

Problem - B - Codeforces

题目大意:给定一个只由0和1组成的序列,序列如果满足si=1,si+1=0,那么可以删除i和i+1种任意一个元素,请求出给定序列进行删除操作之后长度最短且字典序最小的结果.

思路:如果一个串从1开始后面有0又有1最终以0结尾,那么这一段可以看作相同的一部分.最终都可以删掉变成0,那么对于一个字符串的处理就分为三部分,第一部分是遍历掉所有的前置0,因为没有前置1可以和他们匹配,将他们加入答案串,然后使找出中间的10连续段,将他们替换成一个0,然后找出最后一个0,该0以后的串直接加入答案串,然后模拟一下即可.

Problem - C - Codeforces

题目大意:有n个物品,每个商品有自己的价值wi,k个人,每个人需要bi个物品,每个人手上物品的价值等于该人手上价值最大的物品+价值最小的物品,请求出所有人物品价值的最大和.

思路:贪心题,为了使物品价值总和最大,那么就应该尽可能地使所有的价值比较大的物品被选到,对于每个人选择的物品,价值有效的就是他选择的比较大的那个,和最小的那个,为了使较小的那个数尽可能地大,那么就应该先处理手上所持物品较少的人,这样他就可以选择较大的数作为最小值.

Problem - D - Codeforces

题目大意:每个节点的生长满足图一,图二表示的是一个爪子,当一个爪子上所有的节点都未被染色的时候,你可以将它染成黄色,求第n此生长得到的数上最多有多少个节点可以染成黄色.

思路:树形dp,但其实可以直接贪心,每次的抉择一定符合最优解,所以不需要在每个节点处考虑染色或不染色.
