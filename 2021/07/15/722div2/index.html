Problem - A - Codeforces

题目大意:给定一个数组,每次可选该数组的任意个元素求平均值,对于计算的每个元素,如果其数值大于平均值,那么就将其移出该数组.求出给定数组最多可以删除的元素个数.

思路:对于给定序列,最终能留下的元素,一定是数组中最小的元素,所以只需要统计最小元素的个数,然后拿数组总个数减去最小元素个数即可.

Problem - B - Codeforces

题目大意:给定一个序列,对于序列中任意的i<j,都满足:|ai-aj| >= MAX,(MAX是该序列中最大的元素),那么就乘该序列是一个strange序列,先给定你一个序列,请你求出它最长的strange子序列的长度.

思路:首先对于所有的负数和0,都一定可以添加到strange序列中,因为绝对值一定大于等于0,然后考虑正数的情况,最多只能添加一个正数,因为如果有多个正数,那么正数之差的绝对值一定小于较大的那个正数,对于正数的添加,我们还要考虑正数的大小,如果最小的正数已经大于非正数差值的最小值,那么也无法添加.

Problem - C - Codeforces

题目大意:给定n个节点,每个节点有两个值l,r,n个节点有n-1条边互相连接,你可以对于每个节点选择其区间[l,r]上的值作为该节点的值,每两个节点的差值的绝对值代表了这两个节点之间的距离,现在求整棵树最大的距离之和.

思路:树形DP,首先n个节点,n-1条边,这个图一定是一棵树,同时对于每个节点的数值的选择,一定要不是l要不是r,否则复杂度就太高了,(我不会证为什么要取端点).定义dp数组为dp[n][2],n表示对于某个节点,2表示该节点选取l或者r的最大值,然后dfs跑一遍,

Problem - D - Codeforces

题目大意:给定2n个点,你每次选取两个点划出一条线段,每两条线段必须满足这样两种关系的一种:

  1. 两线段长度相等

  2. 一条线段包含另一条线段

请问给定2n个节点最多可以画出多少种线段.

下图中A和B都是合法,C不合法,因为红线没有被蓝线包裹.

思路:dp,首先考虑初始状态,每条线段长度相等的情况,对于n条线段,它的距离从1,2,3…依次枚举,如果线段个数可以被线段长度整除,那么这种就存在一种各线段长度相等的情况.

对于额外增加一条线段,直接将线段放在两边的端点上即可,然后更新一下状态.
