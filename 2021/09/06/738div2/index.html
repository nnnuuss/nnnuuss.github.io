Problem - A - Codeforces

题目大意:你每次可以选取一个区间[l,r],使得对于(0<=i<=r-l)每个ai=al+i&ar-i,你可以进行若干次这种操作,使得给定序列中的最大值尽可能的小.

思路:因为&操作具有连续性,这个操作可以将所有的ai进行&操作,那么最小值就是所有的数都进行&.

Problem - B - Codeforces

题目大意:给定你一个由B,R,?组成的字符串,请你替换掉其中的?,使得RR,BB出现的次数尽可能的少.

思路:简单构造,找到第一个不为?的数,作为起点,让后向两边交替的填R和B.

Problem - C - Codeforces

题目大意:现在有n+1个点,2n-1条有向边,i到i+1有一条有向边(1<=i<=n-1),然后给定n个数,如果给定的是0就表示可以从当前节点到达第n+1个节点,如果是1就表示可以从第n+1个节点到达当前节点,现在请你找出一个遍历顺序,可以遍历整张图,所有的点只遍历一次,可以随意选择起点.

思路:因为从1到n-1是连通的,那么我们就需要想办法将n和其他的节点连接起来,如果a1=1,那么一定符合,直接n作为起点,然后n->1->…->n-1,an=0,那么就可以直接从第n个节点到达第n+1个节点也是符合的,对于其它一种情况就是找01序列,这样从ai到an+1然后再到ai+1.

Problem - D1 - Codeforces

题目大意:有两片森林,每片森林中有若干个节点相连接,现在你可以将两片森林中的某些节点连接起来,每次连接两个节点,两片森林中对应的节点都会被连接,请输出最多可以进行的连接方式.

思路:用并查集维护两片森林,然后暴力枚举连接方式.
