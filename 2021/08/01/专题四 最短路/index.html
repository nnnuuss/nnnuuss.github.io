专题四 最短路

发表于 2021-08-01 更新于 2021-08-03 阅读次数：

Til the Cows Come Home - POJ 2387 - Virtual Judge (vjudge.net)

板子题直接上dijk的板子

Frogger - POJ 2253 - Virtual Judge (vjudge.net)

题目大意：请你求出从从节点1到节点2的路径中，距离最短的一步。

思路：改写最短路的板子，对于从1到2的每个节点不再采取整体最优，而采取局部最优解。

Heavy Transportation - POJ 1797 - Virtual Judge (vjudge.net)

求从1到n的最大生成树的最小边.

Silver Cow Party - POJ 3268 - Virtual Judge (vjudge.net)

双向建图,然后跑两边dijk

Currency Exchange - POJ 1860 - Virtual Judge (vjudge.net)

floyd跑一遍,看是否有正环,也可以用spfa.

Wormholes - POJ 3259 - Virtual Judge (vjudge.net)

floyd求负环,卡时间了

MPI Maelstrom - POJ 1502 - Virtual Judge (vjudge.net)

单源最短路径

Cow Contest - POJ 3660 - Virtual Judge (vjudge.net)

floyd来传递胜负关系,如果一个人输的和赢得加起来达到了n-1场,那么他的分数就确定了.

Arbitrage - POJ 2240 - Virtual Judge (vjudge.net)

依然是floyd求负环

Invitation Cards - POJ 1511 - Virtual Judge (vjudge.net)

双向建边dijk,一直wa心态都崩了直接贴别人的答案了.
