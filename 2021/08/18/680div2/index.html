Problem - A - Codeforces

题目大意:给定序列a和序列b,你可以重新对序列a,b进行排序,问是否能保证对于序列中的每个元素满足ai + bi <= x.

思路:直接将两个序列进行排序,然后序列a由小到大序列b由大到小遍历一次即可.

Problem - B - Codeforces

题目大意:现在有两场比赛,现在给出分数a,代表的是第一场比赛排名为100名的分数,然后给出分数b,分数b代表的是第一场比赛中排名前一百的人在第二场比赛中的最少得分.又给出分数c,代表的是在第二场比赛中排名100名的分数,分数d代表这些人在第一场比赛中得到的最少的分数,请问两场比赛分数之和排在第100的人分数最小是多少.

思路:简单思维题.对于数据a,b,那么所能得到的分数最小值为a+b,对于c,d所能得到的分数最小值为c+d,那么其它的分数都大于等于这两个数,第100人的分数一定是这里面两个数较大的那个.

Problem - C - Codeforces

题目大意:给定数p和q,请你找出最大的数x,使得x是p的因数,且q不是x的因数.

思路:数论题.如果p不能被q整除,那么q就不是p的因数,那么最大的数x=p.如果可以被整除,那我们只需要求出一个最小值,当p除以这个最小值时,q不再是p的因数即可.例如4是p的因数,也是q的因数,那么如果将p除以2以后,q就比p多了一个因数2,q就不再是p的因数了,所以我们需要做的就是找出p和q的所有因数,然后对每个因数进行枚举就可以了.
