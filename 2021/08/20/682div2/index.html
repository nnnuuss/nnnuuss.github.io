Problem - A - Codeforces

题目大意:如果一个序列的任意子序列之和可以被序列的长度整除,就称这个序列为good序列,现在给定长度n,请你构造一个长度为n的good序列.

思路:简单构造.

Problem - B - Codeforces

题目大意:给你一个长度为n的序列b,由序列b生成序列a,ai = 2^bi,请你判断是否存在l1,r1l2,r2,使得

思路:反证法,如果序列a中存在两个数相等,那么一定有解,现在考虑如果没有两两相等的情况下是否有解,没有两两相等的情况,那么就不存在进位的情况,一个数无论怎么加都没办法保证使低位为0,所以唯一解是有两个数相等.

Problem - C - Codeforces

题目大意:给定一个大小为n*m的矩阵a,现在根据矩阵a构造一个矩阵b,,现在要保证矩阵b中相邻的元素不相等.

思路:构造题,从奇偶性考虑,如果矩阵每一个元素和行列之和的奇偶性相同,那这种是可行的,类似黑白棋盘染色.

Problem - D - Codeforces

题目大意:给定一个长度为n的序列,你每次可以选择三个数进行异或操作,判断是否可以通过至多n此操作来使得所有的数相等.

思路:比较容易发现规律的构造题,对于n是奇数的情况,可以按照1,2,3;3,4,5这样的顺序来处理,处理到最后1,2相等,3,4相等,只需要选择1,2,n;3,4,n就可以得到答案.当n是偶数的情况,还是按照同样的思路来处理,最后是多出了一个数,那么为了使答案成立,所有的数异或的结果应该为0,否则就无解.
